== Description ==

Maid Booking plugin adds a simple maid booking slider, which you can add to your post, page or template using [maid_slider] shortcode.

* Send details of booking order to email
* Set base price, price per bedroom/bathroom
* Limit number of bedroom's/bathroom's
* Maid Booking Slider works with half bathroom's.
* Set title, subtitle and button text for the Maid Booking Slider

List of Maid booking slider shortcode attributes, you can set:

* change button text color as [maid_slider color=#72777c]
* change button background color as [maid_slider bg=#46b450]
* set slider maximum width as [maid_slider width=600px]

You can of course combine all those shortcode attributes for example as [maid_slider bg=#4aa400 color=#fff width=400px] which will limit your slider width to 400px with nice green button with white text.

== Installation ==

* Upload the entire maid-booking folder to the /wp-content/plugins/ directory.
* Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Maid booking' menu in your WordPress admin panel under Settings menu.
